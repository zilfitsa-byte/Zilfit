# Z-Patent Live Brief v1

## Project
ZILFIT

## Invention Direction
A full 3D-printed footwear system designed around pressure-informed geometry, recovery-oriented use, controlled sensory stimulation zones, and structured wording boundaries.

## Core Patent Intent
Protect the combination of:
- full-print footwear architecture
- pressure-informed zone geometry
- controlled stimulation / massage ridge placement
- recovery-oriented use context
- variable wall thickness and lattice logic
- wording-safe non-medical product framing

## Must Protect
- geometric zoning logic
- print-specific structural logic
- recovery-use embodiment
- female-specific embodiment
- sensory-stimulation embodiment
- claim boundary discipline

## Must Avoid
- direct disease-treatment claims
- unvalidated hormone-regulation claims
- vague novelty language without boundary
- prior-art overlap with generic insoles or generic reflexology claims
