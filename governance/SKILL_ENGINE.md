# ZILFIT Skill Engine v1

## RULE 1
No agent is allowed to produce raw output.

## RULE 2
Each task MUST call predefined skills before output.

## RULE 3
Skills act as constraints, not optional tools.

## RULE 4
Output must be structured and validated.

## CORE SKILLS

### Research Tasks
- Deep Research Synthesizer
- Source Validation
- Knowledge Structuring

### Design Tasks
- Workflow Automation Agent
- Flowchart Decision Builder
- UI/UX Layout Advisor

### Product Tasks
- Competitive Intelligence
- Knowledge Structuring

### Writing / Reports
- SCQA Writing Framework
- Structured Copywriting

## EXECUTION FLOW

1. Receive task
2. Select skills
3. Apply skills
4. Structure output
5. Validate
6. Save to /reports
